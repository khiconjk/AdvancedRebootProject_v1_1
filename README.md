# AdvancedReboot v1.1

Fixes Recovery reboot permission by adding:

```txt
android.permission.RECOVERY
```

Required privileged permissions:

```txt
android.permission.REBOOT
android.permission.RECOVERY
android.permission.DEVICE_POWER
```

Install as:

```txt
/system/priv-app/AdvancedReboot/AdvancedReboot.apk
/system/etc/permissions/privapp-permissions-advancedreboot.xml
```
